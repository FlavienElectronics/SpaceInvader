\doxysubsection{Detailed Description}
Implementation of the \doxylink{class_point}{Point} class, representing a point in the game world. 

The \doxylink{class_point}{Point} class provides basic functionality for managing a point\textquotesingle{}s position (x, y) and color. It supports operations like moving the point along the x or y axis, hiding the point, and accessing its position and color. This class extends SFML\textquotesingle{}s Rectangle\+Shape to render the point visually.
