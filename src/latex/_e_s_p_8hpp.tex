\doxysubsection{Detailed Description}
Declaration of the \doxylink{class_e_s_p}{ESP} class, used for managing communication via USART. 

This file contains the definition of the {\ttfamily \doxylink{class_e_s_p}{ESP}} class, which facilitates communication between a microcontroller or other device and a computer via the USART (Universal Synchronous/\+Asynchronous Receiver/\+Transmitter) interface. It allows for sending and receiving messages over a serial connection. The class includes functions for checking the connection status, reading data from the serial port, and sending messages. Additionally, it defines a structure {\ttfamily USART\+\_\+package} to store data read from the interface, such as device information, string size, and value.
