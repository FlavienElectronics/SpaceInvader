\doxysubsection{Detailed Description}
Defines the \doxylink{class_letter_group}{Letter\+Group} class and its associated letter classes (A, E, G, M, N, O, R, U, V, W, Y, Exclamation). 

This file provides the declaration of the {\ttfamily \doxylink{class_letter_group}{Letter\+Group}} class, which manages a group of letters drawn on a window. Each letter is represented as a set of points, and the group can render the letters on the screen using SFML. The letter shapes are predefined, and each letter (A, E, G, M, N, O, R, U, V, W, Y, Exclamation) is implemented as a subclass of the {\ttfamily Letter} class.

The {\ttfamily \doxylink{class_letter_group}{Letter\+Group}} class provides functionality to initialize the group with a specific color and window dimensions, draw the group to the screen, and manage the destruction of all letters within the group.
