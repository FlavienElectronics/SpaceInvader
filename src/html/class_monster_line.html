Constructor for the MonsterLine class.

Initializes a new line of monsters at specified position, with a random direction and the given number of monsters. Each monster is allocated dynamically and placed with a certain spacing between them.

Parameters
winPointer to the SFML window used for rendering the game.
windowHeightThe height of the game window.
windowWidthThe width of the game window.
x_posThe initial x-coordinate of the first monster in the line.
y_posThe initial y-coordinate of the monsters in the line.
numberOfMonsterThe total number of monsters in the line.
colorThe color used for the monsters (currently unused in the constructor).
